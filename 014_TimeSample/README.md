# Install
```
npm init -y
npm install -g truffle
npm install -g ganache-cli
npm install -g hardhat
```

or

```
npm install
```

# start ethereum node
```
npx hardhat node
```

# compile
```
npx hardhat compile
```

# deply
```
npx hardhat ignition deploy ./ignition/modules/TimeSample.js --network localhost
```


# test
```
npx hardhat test
```
